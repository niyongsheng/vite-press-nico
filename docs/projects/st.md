## 项目简介
**ST (Smart Canteen)** 智慧食堂是面向园区/企业/学校的数字化食堂管理平台，提供商户、人员、规则、设备与报表的一体化管理，支持线上支付、退款、账户/卡片管理与多维度统计分析。

## 核心功能
- 食堂管理：支付/退款配置，窗口与设备管理，餐别、菜品分类/管理，菜谱管理。
- 人员管理：用户列表、卡片列表、账户列表与账号管理，支持实名认证与权限控制。
- 规则管理：优惠规则与限制规则，可配置折扣、限购、时段等策略。
- 汇总报表：支付记录、退款记录、充值记录，菜品/商家/园区多维统计，支持运营决策。
- 用户协议：服务协议与隐私协议页面，便于合规展示。
- 文档与支持：内置“项目文档”入口。

## 技术栈与前端路由
- 前端：基于 Vue3（典型使用 Layout + 业务组件），模块化路由包含：
  - `/canteen-manage`（食堂管理）
  - `/personal-manage`（人员管理）
  - `/rule-manage`（规则管理）
  - `/report-manage`（汇总报表）
- 后端：采用 Spring Boot + Redis + MyBatis支撑业务数据处理与报表统计。

## 业务价值
- 提效：统一配置支付、窗口、设备和菜品，减少线下维护成本。
- 精细化运营：优惠/限制规则灵活配置，提升就餐体验与客单价。
- 可视化决策：多维报表帮助掌握营收、菜品偏好与园区分布。
- 合规与安全：用户协议与隐私协议内置展示，满足合规要求。

## 典型使用场景
- 园区/企业食堂：分窗口运营、员工刷卡/扫码支付、优惠补贴与报表。
- 学校食堂：餐别管理（早餐/午餐/晚餐），菜品统计与充值记录。
- 多商户档口：窗口/设备独立管理，商家维度的营收与菜品分析。

## 可扩展方向
- 营销：会员积分、满减/套餐等营销玩法。
- 物联网：与称重、闸机、取餐柜等设备联动。
- 数据：对接 BI 或大屏，增加实时监控与预测能力。
- 权限与组织：细粒度角色、部门层级与多园区支持。
