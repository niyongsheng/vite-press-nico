# SKC - 智能钥匙柜管理系统

## 📋 项目概述

**SKC (Smart Key Cabinet)** 是一套企业级智能钥匙柜管理解决方案，专为现代办公、酒店、公寓、学校等场所的钥匙管理需求而设计。系统采用物联网技术，结合智能硬件和软件平台，提供安全、高效、智能的钥匙存取管理服务。

### 🎯 系统特点
- **智能化管理**：AI识别、自动分配、智能预警
- **安全可靠**：多重身份验证、操作记录追溯、异常报警
- **多场景适配**：支持办公、酒店、公寓、学校等多种应用场景
- **实时监控**：24小时实时监控、状态跟踪、远程管理
- **数据分析**：使用统计、效率分析、智能报表

## 🚀 核心功能模块

### 1. 钥匙管理
- **钥匙登记**：支持批量导入、标签管理、分类归档
- **智能分配**：基于权限和需求的智能钥匙分配算法
- **状态跟踪**：实时跟踪钥匙位置、使用状态、归还时间
- **生命周期管理**：从入库到报废的完整生命周期管理

### 2. 用户管理
- **身份认证**：支持刷卡、指纹、人脸识别、手机NFC等多种认证方式
- **权限控制**：细粒度权限管理，支持角色和部门权限设置
- **用户档案**：完整的用户信息管理和操作历史记录
- **访客管理**：临时访客权限申请和审批流程

### 3. 存取管理
- **智能存取**：一键存取、预约存取、批量存取
- **预约系统**：支持提前预约、时间段管理、冲突检测
- **归还提醒**：智能归还提醒、超期预警、自动催还
- **异常处理**：丢失报告、紧急处理、应急预案

### 4. 安全监控
- **实时监控**：24小时视频监控、状态监控、异常检测
- **安全预警**：未授权访问、异常操作、系统故障预警
- **操作日志**：完整的操作记录、审计追踪、合规报告
- **应急响应**：紧急情况处理、远程锁定、安全模式

### 5. 数据分析
- **使用统计**：钥匙使用频率、热门时段、用户行为分析
- **效率分析**：存取效率、等待时间、系统性能分析
- **预测分析**：使用趋势预测、容量规划、优化建议
- **报表系统**：多维度数据报表、可视化图表、导出功能

### 6. 系统管理
- **设备管理**：硬件设备监控、固件升级、故障诊断
- **配置管理**：系统参数配置、业务规则设置、个性化定制
- **备份恢复**：数据备份、系统恢复、灾难恢复
- **系统监控**：性能监控、资源使用、健康状态检查

## 🛠️ 技术架构

### 硬件技术栈
- **主控制器**：ARM Cortex-A72 四核处理器
- **存储模块**：eMMC 32GB + 外置SD卡扩展
- **通信模块**：WiFi 6 + 蓝牙 5.0 + 4G/5G模块
- **传感器**：温湿度传感器、震动传感器、光感传感器
- **执行器**：电磁锁、电机驱动、LED指示灯

### 前端技术栈
- **Web管理端**：Vue 2 + TypeScript + Element/ UI
- **移动端应用**：Uni + uView + colorUI
- **小程序**：微信小程序 + 支付宝小程序
- **状态管理**：Vuex
- **构建工具**：Webpack

### 后端技术栈
- **开发语言**：Java 8
- **服务框架**：Spring Boot 2.x
- **数据库**：MySQL 8.0 + Redis 7.0
- **消息队列**：MQTT
- **缓存**：Redis Cluster

### 物联网技术
- **通信协议**：MQTT + HTTP/HTTPS + WebSocket
- **设备管理**：IoT Hub + 设备注册 + 状态同步
- **数据采集**：实时数据采集 + 批量数据上传
- **边缘计算**：本地数据处理 + 云端协同计算
